# ⚡ ویژگی‌های اصلی

### 🌐 اتصال زنده به Google Sheets

ارتباط مستقیم با دیتابیس بدون نیاز به سرور اختصاصی.

### 🧩 معماری ماژولار

تفکیک کامل ساختار پروژه در فایل‌های مستقل.

### 🕒 ساعت و تشخیص منطقه زمانی

نمایش ساعت زنده و پرچم کشور کاربر.

### 📸 کارت‌های سه‌بعدی

نمایش تصاویر مودها با افکت تعویض لایه‌ای.

### 🏆 لیدربرد ادمین‌ها

رتبه‌بندی پویا براساس تعداد پست‌های ثبت‌شده.

### 🔊 Web Audio API

افکت‌های صوتی بدون نیاز به فایل‌های خارجی.

### 🌈 رابط کاربری Cyberpunk

افکت‌های RGB، Neon Glow و Skeleton Loading.

### 🔍 جستجوی پیشرفته

فیلتر براساس دسته‌بندی، برند، سازنده و تگ‌ها.
