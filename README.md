# 🏎️ IRAN SLIDE ZONE

### 🏁 Premium Assetto Corsa Mod Hub Launcher

---

## 🌟 Overview

Iran Slide Zone is a modern and high-performance platform designed for managing, showcasing and distributing Assetto Corsa mods through a fully cloud-powered architecture.

✨ Live Google Sheets Database

✨ Zero Backend Costs

✨ Cyberpunk UI Design

✨ Advanced Search & Filtering

✨ Admin Leaderboard

✨ Web Audio Effects

✨ 3D Image Flip Animations

✨ Modular ES6 Architecture
